Usage

Create a code

createCode(channel_id, {
    activity: "youtube", //choose one form the list
	/*optionnal*/ duration: 86400, //in ms 
	/*optionnal*/ neverExpire: false,
	/*optionnal*/ maxUses: 0
})

that will return you something like that:

{

    activity: {
        id: 880218394199220334,
        name: "youtube",
        icon: "",
        description: "",
    },
    guild: {...}, //the guild
    channel: {...}, //the channel
    inviter: {...} //the user
}
